# Alpine Linux
## Base Installation
* download iso from internet
* create vm, boot
* run `setup-alpine` script
* `reboot`

## Auto Personalization
* login as `root`
* `apk add git`
* `git clone https://github.com/tobyw121/Alpine-Linux-XFCE4/`
* * `chmod 777 -R Alpine-Linux-XFCE4/`
* `cd Alpine-Linux-XFCE4/`
* `./setup.sh`
